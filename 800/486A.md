# Codeforces 486A - Calculating Function

## Problem Link

https://codeforces.com/problemset/problem/486/A

---

## Idea

The sequence is:


-1 + 2 - 3 + 4 - 5 + ...


I noticed that if I group the numbers in pairs:


(-1 + 2) = 1
(-3 + 4) = 1
(-5 + 6) = 1


Each pair always gives `1`.

---

## Solution

- If `n` is even, all numbers form pairs.

  Example:


  n = 8


  Number of pairs:


  8 / 2 = 4
  ```

  So, the answer is:

 
  n / 2
 

- If `n` is odd, one negative number remains at the end.

  Example:

 
  n = 7
 

 
  (-1 + 2) + (-3 + 4) + (-5 + 6) - 7
 

  So, the answer is:


  -(n + 1) / 2
  

---

## Complexity

- Time Complexity: `O(1)`
- Space Complexity: `O(1)`
