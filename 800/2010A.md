# Codeforces 2010A - Alternating Sum of Numbers

## Problem Link

https://codeforces.com/problemset/problem/2010/A

---

## Idea

The required sum is:


a1 - a2 + a3 - a4 + ...


So, while traversing the array, use:

cpp
a[i] = (i % 2 != 0) ? -a[i] : a[i];


- If `i` is odd, make `a[i]` negative.
- Otherwise, keep it unchanged.

This way, we can simply add all the elements at the end.

---

## Example


1 2 3 4


becomes


1 -2 3 -4


Sum:


1 - 2 + 3 - 4 = -2


---

## Complexity

- Time Complexity: `O(n)`
- Space Complexity: `O(1)`
