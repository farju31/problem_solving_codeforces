# Codeforces 1742A - Sum

## Problem Link

https://codeforces.com/problemset/problem/1742/A

---

## Idea

Check whether the sum of any two numbers is equal to the third number.

### Conditions


a + b == c


or

a + c == b


or


b + c == a


If any condition is true, print `YES`; otherwise, print `NO`.

---

## Complexity

- Time Complexity: `O(1)`
- Space Complexity: `O(1)`
